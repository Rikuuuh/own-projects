    Smart Living Community App

    As my educational capstone project, we developed a mobile application for housing cooperatives as a team of four using Flutter. The project includes both frontend & backend development, as well as the design of a database (Firebase). The app features functionalities for user login, updating user information, viewing and deleting personal bookings, a reservation calendar (for sauna slots, laundry times, and guest parking), and notice boards (where users can post questions etc.).
    My responsibilities in this project were focused on implementing the user registration and login processes, developing the navigation drawer, managing the functionality of the reservation calendar, and initializing and utilizing the database.
